📊 Market Sentiment vs Trader Behavior

📌 Overview
This project analyzes how market sentiment (Fear vs Greed) impacts trader behavior, profitability, and risk-taking.

⚙️ Setup

Install required libraries:

pip install pandas matplotlib

▶️ How to Run

1. Open the notebook in Google Colab or Jupyter
2. Upload both datasets:
   - fear_greed_index.csv
   - historical_data.csv
3. Run all cells sequentially


📈 Key Findings

- Traders earn higher profits during Greed phases
- Trading activity increases in bullish markets
- Fear leads to reduced participation


💡 Strategy Recommendations

- Reduce leverage during Fear markets
- Increase trade activity during Greed with controlled risk

📁 Files Included

- analysis.ipynb (main notebook)
